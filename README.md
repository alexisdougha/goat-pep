# goat-pep

**goat-pep** is a small utility repository to convert and navigate between **PDB** and **XYZ** files for peptide-focused quantum chemistry workflows, in particular when running **GOAT conformer searches with ORCA 6**.

The main use case is:

1. Start from peptide structures generated by [**PEP-EDIT**](https://pep-edit.rpbs.univ-paris-diderot.fr/)
2. Prepare ORCA input files for GOAT conformer searches
3. Convert ORCA-generated XYZ conformations back to PDB for downstream structural analysis if needed

---

## Installation

Clone the repository:

```bash
git clone https://github.com/alexisdougha/goat-pep.git
cd goat-pep
```

(Optional) Create and activate a virtual environment if needed but it should work with any version of Python3

---

## Preparing ORCA GOAT inputs from PEP-EDIT outputs

* ORCA GOAT documentation: [https://www.faccts.de/docs/orca/6.0/tutorials/prop/goat.html](https://www.faccts.de/docs/orca/6.0/tutorials/prop/goat.html)
* GOAT paper: [https://onlinelibrary.wiley.com/doi/10.1002/anie.202500393](https://onlinelibrary.wiley.com/doi/10.1002/anie.202500393)

Starting from the **PDB** and **SMILES** files generated by **PEP-EDIT**, use:

```bash
python pep-edit_to_orca.py \
    data/pep-edit_to_orca/inputs/pep-edit_structure.pdb \
    data/pep-edit_to_orca/inputs/pep-edit_smiles.txt \
    data/pep-edit_to_orca/outputs
```

This script generates:

* `.xyz` coordinate files
* `.inp` ORCA input files suitable for GOAT conformer searches

---

## Converting ORCA XYZ conformations back to PDB

After running ORCA, you may want to analyze the resulting conformations with PDB-compatible tools.

Use:

```bash
python orca_xyz_to_pdb.py \
    data/orca_xyz_to_pdb/inputs/pep-edit_structure.pdb \
    data/orca_xyz_to_pdb/inputs/orca_conformations.xyz \
    data/orca_xyz_to_pdb/outputs
```

This maps the ORCA-generated XYZ conformations onto the original PDB topology and outputs PDB files for downstream analysis and visualization.
